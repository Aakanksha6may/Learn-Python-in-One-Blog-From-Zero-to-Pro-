🛠️ 1. Setting Up: Python & VS Code (Step-by-Step)

✅ Step 1: Download Python

  1. Visit: https://www.python.org/downloads
  2. Click the yellow Download Python button
  3. Check the box “Add Python to PATH”
  4. Click "Install Now"

🎉 Done. Python is now on your system.

✅ Step 2: Download VS Code

  1. Visit: https://code.visualstudio.com/
  2. Download for your OS and install

🔌 Install Python Extension:

  1. Open VS Code
  2. Go to Extensions (Ctrl+Shift+X)
  3. Search "Python" by Microsoft and install

✅ Step 3: Test it!

print("Hello, world!")

Right-click in VS Code > Run Python File in Terminal

Hello, world!

🔁 3. Control Flow (If/Else, Loops)

⚠️ 6. Error Handling

try:
    num = int(input("Enter a number: "))
    print(10 / num)
except ZeroDivisionError:
    print("Cannot divide by zero.")
except ValueError:
    print("Not a valid number.")
finally:
    print("Done trying!")

👑 7. Classes & Objects

        self.name = name

    def bark(self):
        print(f"{self.name} says Woof!")

my_dog = Dog("Bruno")
my_dog.bark()
